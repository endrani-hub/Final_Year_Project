Web-based Application for Gender Detection , Mask Detection and Fake Human Face Generation.

This is the Final Year project curated by Indrani Roy(16500317051) , Arijit Ghosh(16500317065) and Anirban Bej(16500317071)

This project makes use of the Deep Learning field to curate a modern multipurpose web-app. The entire project is done with the purpose of providing a real-world application as well as depict a viving and friendly implementation. The creation of the Gender and the Mask detection system entirely resonates the modern time which is suffocated with the curse of CoVID-19. Both of these projects are built with the help of Convolutional Neural Networks on top of the famous Tensorflow framework. The Fake Human Face Generator model makes use of Generative Adversarial Networks and is built with the help of Pytorch Framework.
